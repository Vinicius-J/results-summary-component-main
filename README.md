## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->
`08_results-summary-component-main` é um projeto criado ultilizando como referência modelos já prontos do [Frontend Mentor](https://www.frontendmentor.io/).

O objetivo maior com esse projeto era ganhar experiência e propor um desafio a mim mesmo na criação de uma página web do início.
